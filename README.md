# Scientific Calculator (C++)

A beginner-friendly **scientific calculator** built in **C++** (console/terminal).

## Features
- Basic arithmetic: add, subtract, multiply, divide
- Power (x^y)
- Square root
- Trig functions: sin, cos, tan (radians)
- Logs: log10 and ln
- Factorial (0..20)
- Input validation (no crashing on bad input)
